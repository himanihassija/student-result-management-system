STUDENT RESULT MANAGEMENT SYSTEM (PYTHON)

PROJECT DESCRIPTION
The Student Result Management System is a Python-based console application used to manage student academic records.
It allows users to add student details store subject-wise marks calculate total percentage and grade automatically.
The system uses JSON file storage to ensure data persistence.
This project is suitable for beginner to intermediate Python learners and for college mini project submission.

TECHNOLOGIES USED
Python 3
JSON for data storage
File handling
Core Python concepts

FEATURES
Add new student records
Store marks for multiple subjects
Automatic calculation of total percentage and grade
View individual student results
View all student records
Update existing student marks
Persistent data storage using JSON
Menu driven console interface

PROJECT STRUCTURE
Student Result Management System
student_result_management.py
students.json
README.md

NOTE
The students.json file is created automatically when the program is run for the first time.

HOW TO RUN THE PROJECT
Install Python 3 on your system
Download or clone the project folder
Open terminal or command prompt in the project directory
Run the command python student_result_management.py
